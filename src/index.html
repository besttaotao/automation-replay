UniversalMacroRecorder

就绪

点击“开始录制”完成一次操作；录制结束后会弹出保存名称，保存后即可执行。

F8 停止录制 F9 停止回放
1 录制点击开始录制，倒计时后去目标页面操作。
2 自动保存录制结束后输入名称，模板固定保存到 macros。
3 选择执行从模板列表选中 JSON，确认次数后执行。

1录制动作

点击开始录制后有 3 秒倒计时。倒计时结束后，切到目标页面完成一次完整操作。完成后按 F8 停止录制，并输入模板名称保存。

2模板列表

这里自动读取固定目录 macros 下的 JSON 模板。点击某个模板后，会自动加载为当前执行模板。

3执行模板

先选择模板，再设置循环次数和步骤间隔。间隔按秒计算，用来适应网站加载和网络波动。
