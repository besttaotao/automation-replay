# UniversalMacroRecorder

### 软件介绍

UniversalMacroRecorder 是一款面向 Windows 桌面环境的通用键鼠录制回放工具。它可以录制用户的鼠标点击、滚轮、键盘输入等操作，并保存为本地 JSON 模板，后续可按指定循环次数自动回放。

本工具适合在网页、桌面软件和固定流程场景中复用重复操作。第一版采用屏幕绝对坐标方案，使用时请尽量保持窗口位置、分辨率和系统缩放一致。

### 核心功能

- **全局键鼠录制**：支持录制鼠标点击、滚轮、键盘按下/抬起等行为。
- **模板化保存**：录制完成后可命名保存为 JSON 模板，统一存放在 `macros` 目录。
- **模板管理**：自动读取模板列表，支持选择加载、刷新列表和删除模板。
- **自定义回放**：支持设置循环次数、无限循环和固定步骤间隔，适配网页加载与网络波动。
- **实时日志**：录制和回放过程中实时输出鼠标坐标、键盘命令和执行状态。
- **倒计时提示框**：录制/回放开始前显示倒计时交互框，关闭即可取消当前操作。
- **窗口调度**：倒计时结束后自动最小化程序；录制停止或回放结束后自动恢复到前台。
- **安全停止**：录制按 `F8` 停止，回放按 `F9` 停止，也可通过界面按钮停止。

### 使用说明

1. 下载 Release 中的 `UniversalMacroRecorder-0.1.0-x64.exe`。
2. 双击运行程序，无需安装。
3. 点击“开始录制”，倒计时结束后完成一次目标操作。
4. 按 `F8` 停止录制，选择保存或不保存本次模板。
5. 在模板列表中选择要执行的 JSON 模板。
6. 设置循环次数和步骤间隔秒数，点击“启动执行”。
7. 执行中可按 `F9` 或点击“停止执行”终止回放。

### 开发与打包

开发模式：

```powershell
npm install
npm run start
```

打包：

```powershell
npm run dist
```

生成的 exe 位于 `dist` 目录。

### 注意事项

- 当前版本使用屏幕绝对坐标，回放前请保持窗口位置、分辨率、缩放比例一致。
- 不建议录制密码、验证码、支付等敏感操作。
- 模板文件为本地明文 JSON，请自行管理敏感数据风险。
- 如果网页存在加载延迟，请适当调大“步骤间隔（秒）”。
